# Lumen Warm

## Product Purpose
Lumen 是一个 AI 桌面应用（Python FastAPI + Tauri 2 + React），面向独立创作者/开发者，用于创意写作、角色扮演、知识管理。目标是专业级产品。

## Users
独立创作者/开发者。使用场景：长时间深度对话、沉浸式写作、知识库管理。需要专注力和沉浸感，不希望被刺眼或花哨的界面打断思路。

## Brand Personality
**温暖 · 精致 · 手作**
- 温暖：配色偏暖调（赤陶棕/暖灰），交互有触感反馈，不冰冷
- 精致：每个像素都有意义，间距、字号、色差都经过推敲
- 手作：不是模板感，像匠人打磨的产品，有自己的审美态度

## Register
product

## Color Strategy
**Restrained**：强调色赤陶棕 #CC7C5E 克制使用（10% 面积原则）。90% 中性暖灰承载视觉，强调色只在关键时刻出现。

## Theme
暗色暖灰主题。底色 #141413 → #1F1F1E → #333230 暖灰渐变，不是冷灰。Soft UI 阴影系统：软硅胶质感，按钮微微凸起，按下沉入。

## Aesthetic Reference
Claude Desktop 暖灰配色 + Linear 暗色精致度。手作感而非模板化。

## Design Principles
1. **克制用色**：强调色只在关键时刻出现，中性暖灰承载 90% 视觉
2. **触感层次**：通过 Soft UI 阴影让每个可交互元素有实体感
3. **微妙差异**：相邻板块用 2-4% 明度差区分，不用硬边框
4. **聚焦内容**：UI 退让，消息和创作内容永远是主角
5. **手作感**：避免模板化，每个组件都像精心打磨过的独立作品
6. **NC 参考而不复制**：借鉴 NC 的布局模式，不改 Lumen 的色彩身份

## Anti-References
- 不要 cyan-on-dark / 紫蓝渐变 / 霓虹点缀（AI slop 标志）
- 不要 side-stripe 色条（border-left > 1px）
- 不要渐变文字
- 不要玻璃态泛滥
- 不要所有板块颜色完全一样
