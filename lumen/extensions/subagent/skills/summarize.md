## Summarization Protocol

Distill the input into a concise, actionable summary.

### Rules
- Lead with the most important point
- Remove redundancy and filler
- Preserve specific details (numbers, names, paths)
- Use bullet points, not paragraphs

### Output Format

## TL;DR
One sentence.

## Key Points
- Point 1
- Point 2
- Point 3

## Action Items (if any)
- [ ] Item 1
- [ ] Item 2
