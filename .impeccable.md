# Lumen Design Context

## Users
独立创作者/开发者，使用桌面 AI 应用进行创意写作、角色扮演、知识管理。
使用场景：长时间深度对话，需要沉浸感和专注力，不希望被刺眼或花哨的界面打断思路。

## Brand Personality
**温暖 · 精致 · 手作**
- 温暖：配色偏暖调（琥珀棕/奶油白），交互有触感反馈，不冰冷
- 精致：每个像素都有意义，间距、字号、色差都经过推敲
- 手作：不是模板感，像匠人打磨的产品，有自己的审美态度

## Aesthetic Direction

### 暗色主题（当前）
- 底色暖灰（#141413 → #1F1F1E → #333230），不是纯冷灰
- 强调色赤陶棕 #CC7C5E，克制使用（10% 面积原则）
- 每个「板块」之间有微妙色差（2-4% 明度差），避免同质化
- Soft UI 阴影系统：软硅胶质感，按钮有微微凸起感，按下有沉入感
- 参考：Claude Desktop 的暖灰配色 + Linear 的暗色精致度

### 浅色主题（规划中）
- 底色暖奶油白（#FDFBF7），像羊皮纸/奶泡色
- 面板用浅卡其（#F5F0E8），文字深棕（#3D3832）
- 强调色赤陶棕保持一致（可能微调明度适配浅底）
- 每个「板块」之间仍然保持微妙色差（通过阴影深度和底色微调实现）
- 阴影转为暖色调，不用冷灰阴影

### 质感系统
- 面板级：soft-panel（外凸阴影，1px 微亮边）
- 条目级：soft-item（轻柔凸起）
- 按钮级：soft-btn（触感反馈）→ soft-pressed（按下沉入）
- 所有阴影偏暖调（高光侧用 rgba(60,58,55,...) 而非纯灰白）

### Typography
- 代码：Fira Code / Cascadia Code / JetBrains Mono（等宽回退链）
- UI 文字：系统字体栈（-apple-system, Segoe UI, ...），暂不加载 Web 字体
- 字号阶梯：text-[10px] → text-xs → text-sm → text-base → text-xl，比例 ≥ 1.25
- 标题不使用大写，品牌名用 tracking-widest 小写

### Layout
- NavRail（左 w-56）+ ContentRegion（flex-1）+ RightRail（右 w-10）
- DebugDrawer 等面板从 RightRail 左侧展开（浮动层，不占布局位）
- 输入框「驾驶舱」：textarea + 分割线 + 工具栏在同一个容器内
- 间距阶梯：4px 基准，常用 8/12/16/24/32

## Anti-References
- 不要 cyan-on-dark / 紫蓝渐变 / 霓虹点缀（AI slop 标志）
- 不要 side-stripe 色条（border-left > 1px）
- 不要渐变文字
- 不要玻璃态泛滥（blur 效果仅在浮动面板使用）
- 不要所有板块颜色完全一样（需要微妙色差区分层次）

## Design Principles
1. **克制用色**：强调色只在关键时刻出现，中性暖灰承载 90% 视觉
2. **触感层次**：通过 Soft UI 阴影让每个可交互元素有实体感
3. **微妙差异**：相邻板块用 2-4% 明度差区分，不用硬边框
4. **聚焦内容**：UI 退让，消息和创作内容永远是主角
5. **手作感**：避免模板化，每个组件都像精心打磨过的独立作品
